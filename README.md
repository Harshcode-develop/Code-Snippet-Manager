# AI Powered Code-Snippet-Manager

- Built a responsive, full-stack web application for code snippet organization and interaction. 
- Integrated Gemini AI to provide intelligent features like code correctness checks, automated comments, and direct technical chat, showcasing proficiency in modern web development (React/Express) and advanced API integration.

- Tech Stack used:
  1. Backend: Express.js, MySQL, Node.js
  2. Frontend: React.js, Tailwind CSS
  3. Gemini AI
